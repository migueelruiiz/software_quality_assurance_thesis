# software_quality_assurance_thesis

## Overview
This project presents the implementation of a software quality plan within a real company environment. The work focuses on establishing the foundations for an automated testing framework to improve software reliability and maintainability.

## Objectives
- Design and implement a software quality strategy  
- Introduce automated testing practices  
- Improve test coverage and defect detection  
- Establish scalable QA foundations

## Contributions
- Defined a structured quality plan for the organization  
- Implemented the initial automated testing framework  
- Improved testing workflow and quality practices  
- Documented the full QA adoption process
